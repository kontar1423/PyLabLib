# Лабораторная работа №4 — «Библиотека»

Реализована модель библиотеки с пользовательскими коллекциями и псевдослучайной симуляцией (вариант 1 из ТЗ).

## Что есть
- `src/models.py` — модель книги `Book` с осмысленным `__repr__`.
- `src/collections.py` — списковая коллекция `BookCollection` и словарная `IndexDict` с итерацией, срезами, удалением и проверкой наличия.
- `src/library.py` — класс `Library`, хранит коллекцию книг и индексы, поиски по автору/году/жанру.
- `src/simulation.py` — события симуляции и `run_simulation(steps, seed)` с выводом лога.
- `src/constants.py` — стартовые книги и значения по умолчанию.
- `src/main.py` — точка входа, спрашивает шаги и seed, запускает симуляцию.
- Тесты: `tests/test_library.py`.

## Как запустить
```bash
python3 -m src.main
```
Можно просто жать Enter, чтобы взять значения по умолчанию.

## Как проверить тесты
```bash
python3 -m pytest
```
